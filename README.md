# Sprite

A pixel-art sprite editor. Users organize their work into projects and groups, and draw sprites on a pixel grid in the browser. 
Live on https://ercie.pl

## Tech stack

Frontend
  - React
  - Typescript
  - HTML
  - CSS
  - Vite
  - Zod
  - React Hook Form

Backend
  - C#
  - ASP NET Core Web API
  - EntityFramework Core
  - SQL Server
  - Docker

Hosting (Google Cloud platform)
  - Firebase Hosting - web app
  - Cloud Run - rest api
  - Cloud SQL - database
